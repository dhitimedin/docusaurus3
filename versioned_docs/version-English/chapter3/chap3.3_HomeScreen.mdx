## 3.3.2. Pop-up Notifications
**Functionality**
* Upon logging in, the user may see one or more automatic pop-ups.
* These pop-ups notify the user about outstanding tasks which require attention.
* The notifications may also alert the user of specific urgent issues like coronavirus precautions.

**Page Details**

## 3.3.3. Notifications Icon
**Functionality**
* Shows the number of active Customer Service Requests and Meter Release Requests.
* Tapping navigates to a detailed list of requests.
* This icon is present on most pages.

**Page Details**

<figure>
<br clear="right"/>
<img align="left" src={useBaseUrl("img/scrnshts/3.3_6_NotificationIcon.png")} alt="Home Screen" width="45%"/>
<Clear>

1.  <u>Notification Icon</u>: shows the total number of pending requests; tapping the icon presents the following options
2.  <u>Customer Service Requests</u> redirects to the service request section when tapped <a href="./Consumers#357-consumer-service">(Section 3.5.7)</a> 
3.  <u>Meter Release Request</u> redirects to the Unreleased Meter Section when tapped <a href="./Meter#383-unreleased-meter">(Section 3.8.3)</a> 

</Clear>
<br clear="both"/>
</figure>
